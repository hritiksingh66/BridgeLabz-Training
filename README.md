# BridgeLabz-Training

## Object-Oriented Programming Practice
This repository contains Java class examples demonstrating core OOP concepts:

- **BankAccount**: Banking operations (deposit, withdraw, balance)
- **CartItem**: Shopping cart management
- **Circle**: Geometry calculations
- **Employee**: Employee information system
- **HandBook**: Book management
- **Items**: Inventory management
- **MobilePhone**: Phone specifications
- **MovieTicket**: Ticket booking system
- **PalindromeChecker**: Text palindrome validation
- **StudentsGrades**: Grade calculation system
